## How To Use
>
> ### Clone The repository
> ```bash
> git clone https://github.com/AdithyanA2005/
> ```
>
> ### Install Dependencies
> ```
> npm install
> ```
>
> ### Start Developement server (This may take some time)
> ```bash
> npm start
> ```
>
> ---
>
> ### Open In Browser
> ```
> http://localhost:3000
> ```
